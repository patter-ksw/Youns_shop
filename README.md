# Youns Shop (윤스 샵)

쿠팡, 네이버 스마트스토어, CJ Mall 등 국내 유명 인터넷 구매 사이트의 인기 상품 정보와 선호도를 실시간으로 모집하여 정렬하고, 상세 정보를 확인하여 직접 구매 사이트로 연결해주는 스마트 쇼핑 랭킹 aggregator 웹 어플리케이션입니다.

## 주요 기능
* **실시간 판매 순위 집계**: 쿠팡, 네이버 스마트스토어, CJ Mall의 판매 실적과 소비자 선호도를 30분 단위로 취합하여 랭킹으로 표시합니다.
* **카테고리별 분류**: 패션의류/잡화, 뷰티, 출산/유아동, 식품, 주방용품, 생활용품, 가전디지털, 스포츠/레저 등 세부 카테고리별 랭킹 조회를 지원합니다.
* **실시간 가격비교 및 상세 정보**: 상품 클릭 시 할인율, 할인 전 가격, 별점 및 리뷰 수, 이번주 판매량 등의 상세 메트릭을 다이내믹하게 조회할 수 있습니다.
* **구매 사이트 연동**: 구매하기 버튼 클릭 시 로딩 애니메이션과 함께 실제 제휴/상품 판매 사이트로 신속하고 안전하게 리다이렉트합니다.
* **주기적 순위 변동 업데이트**: 실시간 백라우팅 데이터 동기화 시뮬레이터를 통해 10초 간격으로 상품 판매량 증가 및 랭킹 스왑이 시각적인 플래시 효과와 함께 갱신됩니다.
* **반응형 웹 UI (PC & Mobile)**: 모바일 뷰와 PC 뷰 각각 최적화된 반응형 레이아웃을 통해 기기별로 최상의 UX를 선사합니다.
  * **PC 뷰**: 고정 좌측 사이드바 카테고리 메뉴, 4열 그리드, 대형 배너
  * **모바일 뷰**: 가로 스크롤형 상단 카테고리 칩셋, 2열 컴팩트 그리드, 간소화된 헤더

## 기술 스택
* **Frontend**: HTML5, CSS3 (Vanilla Style CSS), JavaScript (Vanilla ES6)
* **Hosting / Deployment**: Vercel 연동 빌드 지원

## 설치 및 로컬 실행 방법
이 프로젝트는 별도의 컴파일 과정이 필요 없는 순수 정적 웹 프로젝트입니다.

1. 이 저장소를 클론합니다:
   ```bash
   git clone https://github.com/patter-ksw/Youns_shop.git
   ```
2. `index.html` 파일을 브라우저로 직접 실행하거나, Live Server 등의 가벼운 개발 서버를 이용하여 로컬에서 확인하실 수 있습니다.
